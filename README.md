# Portfolio Website

This is my personal portfolio website built using React and Vite. It showcases my skills, projects, training, internship experience, and contact details.

## Features

- Responsive and modern UI
- Sidebar navigation for smooth scrolling
- Projects section with GitHub links
- Internship and Certificates sections
- Training section with project details
- Contact section for communication

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript
- HTML
- CSS

## How to Run Locally

1. Clone the repository
   git clone https://github.com/Yash11tk/Portfolio.git

2. Navigate to the project folder
   cd Portfolio

3. Install dependencies
   npm install

4. Run the project
   npm run dev
